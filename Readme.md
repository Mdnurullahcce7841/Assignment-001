# AI Startup Showcase – DevConf 2026

## Overview

The **AI Startup Showcase** is a custom section created for the DevConf 2026 website as part of the AI Challenge. It highlights innovative AI startups participating in the conference and provides visitors with a quick overview of their products and technologies.

## Features

* Responsive card-based layout
* Startup logo and name
* Short startup description
* Technology tags (e.g., AI, Cloud, React, Python)
* "Visit Booth" call-to-action button
* Modern hover animation for each card
* Clean design that matches the conference theme

## Why This Section?

Instead of using a common section such as FAQ or Newsletter, I chose an **AI Startup Showcase** because it aligns with the conference's focus on technology and innovation. It gives attendees an opportunity to discover emerging startups, explore their products, and connect with their teams during the event.

## Technologies Used

* HTML5
* CSS3 (Grid, Flexbox, Hover Effects)
* Font Awesome (for icons, if used)

## AI Prompt Used

> Design a modern, responsive **AI Startup Showcase** section for a DevConf 2026 website using HTML and CSS. Create a clean card-based layout with six startup cards. Each card should include a startup logo, startup name, a short description, technology tags, and a "Visit Booth" button. Use a professional developer conference theme with subtle hover animations, rounded corners, and responsive spacing.

## AI Tool Used

* ChatGPT

## Note

The AI was used to generate the initial idea, layout, and code suggestions. The final implementation, styling, customization, and integration into the website were completed and adjusted to fit the overall design of the DevConf 2026 project.
